Welcome to Homepage - Microservice

This is our final year project which demonstrates the concept of microservices and how every component or service serves its intent to make a whole project in a distributed manner.

All the above services are stored on independent containers/pods on ECR(Elastic Container Registry) and deployed on Kubernetes Cluster(EKS) which is the objective of the project.

All the code will be stored on GitHub(SCM) repository and upon code commit/manual deployement, the purpose of DevOps is acheived.
